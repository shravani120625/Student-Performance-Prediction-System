\#📊 Student Performance Prediction System (EdTech Analytics Platform)



An AI-powered full-stack EdTech analytics system that predicts student academic performance using Machine Learning and visualizes risk insights through an interactive dashboard.



\---



\## 🚀 Project Overview



This project predicts whether a student is at risk of poor academic performance based on:



\- Attendance  

\- Study hours  

\- Quiz performance  

\- Assignment scores  

\- Past academic performance  



It generates:

\- Risk prediction (LOW / MEDIUM / HIGH)

\- Probability score

\- Explainable insights

\- Visual dashboard analytics



\---



\## 🎯 Objective



To help educational institutions:

\- Identify at-risk students early

\- Improve academic performance

\- Reduce dropout rates

\- Enable data-driven decisions



\---



\## 🧠 Machine Learning Model



\- Algorithm: Classification (Logistic Regression / Random Forest)

\- Input Features:

&#x20; - Attendance

&#x20; - Study Hours

&#x20; - Quiz Average

&#x20; - Assignment Average

&#x20; - Past Score



\- Output:

&#x20; - Risk Probability

&#x20; - Risk Level

\## ⚙️ Installation



\### Backend



```bash

pip install -r requirements.txt

uvicorn api.main:app --reload

````



Backend runs at:

\[http://127.0.0.1:8000](http://127.0.0.1:8000)



\---



\### Frontend



```bash

cd web

npm install

npm run dev

```



Frontend runs at:

\[http://localhost:3000](http://localhost:3000)
